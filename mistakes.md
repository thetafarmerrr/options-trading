## 2026-06-29 | 玉米 C2609 买跨式 — USDA 事件实验

- 买 C2320：bid 39.5/ask 40.0 → 成交 @ 38.5（¥385）
- 买 P2320：bid 27.5/ask 28.0 → 成交 @ 29.0（¥290）
- 总成本 ¥675。逻辑：IV 低 + 事件 → 买跨式赌波动率上升

- 报告利多但中等意外（库存 < 预期 2.1%）。期货仅涨 2 点
- 卖 C2320 @ 31.5，卖 P2320 @ 28.0。回收 ¥595
- 实现亏损 ¥80（-12%）

### 亏损拆解
- IV 微跌 → Vega 正敞口亏约 ¥35-70
- 持仓两天 Theta → 约 -¥12。Gamma 几乎为零

### 错在哪

- 价差成本占了 3%（C 1.25% + P 1.8%）

### 以后
- 事件前做买方先评估：意外程度预期多大？价差吃多少利润？不够不做
- 中等意外撑不起跨式——这是训练 C 里那题的标准答案

## 2026-07-02 | PTA 近值信号 + 极端行情

**发生**：PTA 2609 卖 P5600/买 P5500，6/29 进场。推断期货 5600→5642，实际 5642，卖腿 OTM 仅 0.7%。3 个交易日期货从 5642 跌到 5332（-5.5%），双深实值平仓亏 ¥235，恰好最大亏损。

**错在哪**：
1. 进场时卖腿离期货不到 2%——近值信号。扫描器当时未加近值过滤，推断价比实际低约 42 点，掩盖了真实 OTM 距离
2. 期货单边 5.5% 跌幅是尾部事件级别。信用价差持有期间若遭遇极端方向波动，封顶亏损也会在几天内兑现

**以后**：
1. 近值过滤已在 7/1 加入扫描器——卖腿 OTM < 2% 不交易
2. 持仓期间每天看一次期货，不只盯卖腿行权价。异常波动时无论浮亏是否到最大都评估是否提前平仓
3. 下单前同花顺确认期货现价 > 推断价，不只看 scanner 数字

### 平仓教训：深实值分步平 vs 同时平

平仓时期货 5348，先买回 P5600 @ 370.5，然后卖 P5500 时期货反弹到 5378 → P5500 价格 297→277，少收 ¥100/吨。

深实值区两条腿 Gamma 不一样大——更高行权价的卖腿 Gamma 更大，期货反弹时跌得更快。分步平 = 卖腿已锁高价，买腿被反弹压低 → 净支出更大。同时平 = 卖腿少付的钱会部分抵消买腿少收的钱。

规则：深实值平仓两条腿同时挂。期货在跌立刻平。期货刚反弹等稳定后同时平。

**发生**：玉米 C2611 卖 P2280/买 P2260，期货从 2326 跌到 2294。卖腿只差 14 点（0.6%），Gamma 已在加速。买回卖腿 @ 42 + 卖出买腿 @ 31，平仓净亏 ¥20。

**错在哪**：持有期间未监控卖腿距离。14 点差一天正常波动就能穿——发现时已在边缘。

**以后**：持仓每日检查，卖腿离期货 < 2% 时评估提前平仓。近值 + 到期尚远 → 可以等。近值 + 临近到期 → 平。
