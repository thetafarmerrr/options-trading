## 每日流程（雷打不动）

### 开盘日

1. **持仓检查**：有持仓 → 期货有没有跌近卖腿？浮亏多少？触发止损吗？
2. **跑 scanner**：`python3 tools/unified_scanner.py` → 输入所有品种期货现价
3. **看信号**：信用价差详情里找无事件标记 ⚠️ 的品种
   - 下单前验证：同花顺确认期货现价 > 卖腿行权价
   - 用文华财经组合单，不分腿
4. **跑 iv_collector**：`python3 tools/iv_collector.py`
5. **训练**：B（价差判断）每天 15 题 + 轮转模块 1 个
6. **日志**：`journal/YYYY-MM-DD.md` 填模板
7. **commit**：`git add -A && git commit -m "Day X: ..." && git push`

### 非交易日

1. 训练 B + 轮转模块
2. 读 Natenberg/Sinclair
3. 英语跟读 + 日记
4. 日志 + commit

## 下单决策树（打印贴屏幕旁）

```
scanner 出信号
    │
    ├─ ⚠️有事件标记？ → 不做（或买跨式，L2 后）
    │
    ├─ 卖腿行权价 > 同花顺期货现价？ → 不做
    │
    ├─ 净权利金 ≤ 0？ → 不做
    │
    ├─ 最大亏损 > 本金 5%？ → 不做
    │
    └─ 全通过 → 文华财经组合单 → 下单 → 记日志
```

---

## 止损规则

```
信用价差持有期间：
- 期货跌近卖腿行权价（< 1%） → 全平
- 两腿都变实值 → 亏损已封顶 → 平仓释放保证金
- 事件突然降临（如贸易战新闻） → 立刻平
- 其他情况 → 不动。等到期
```

## 紧急情况处理

| 情景 | 行动 |
|------|------|
| 连续亏损 5 笔 | 停 3 天。不回测。只跑训练。复盘那 5 笔 |
| 连续 10 天 0 信号 | 正常。等。不做 |
| 保证金不够了 | 减仓到总风险 ≤ 10% 本金 |
| 心态崩了 | 停 1 周。不看盘。不内疚 |
| 策略连续 20 笔胜率 < 50% | 这个策略可能不 work。搁置。换品种或换策略 |
