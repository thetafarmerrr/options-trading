## 英文学术搜索指南

### 搜索引擎
- **Google Scholar** → 学术论文和 IMF/美联储工作论文
- **YouTube** → panel discussions, interviews, documentary
- **SSRN** → 最新金融研究预印本
- **FRED (St. Louis Fed)** → 宏观经济数据

### 常用搜索模式
```
"event name" + tail risk / volatility / options
"event name" + VIX / implied volatility
"event name" + commodities / futures
"event name" + post-mortem / lessons learned
```

### 示例
```
"Volmageddon" VIX tail risk        → 波动率策略爆仓分析
"negative oil price" post-mortem   → WTI 负价格复盘
"yen carry trade unwind" Nikkei    → 2024 年 8 月日元套利
"Flash Crash 2010" HFT             → 高频交易与市场结构
```

### 每个事件的学习模板
1. 搜 1 篇 Wikipedia 词条（快速了解全貌）
2. 搜 1 篇 post-mortem 文章（Bloomberg/FT/Reuters）
3. 搜 1 个 YouTube 采访/纪录片（练听力）
4. 记三个关键词：`event name` `market impact` `lesson learned`
