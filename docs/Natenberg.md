## 第 2 章：期权定价基础

### 期权价格由什么决定

| 因素 | 变化方向 | Call 价格 | Put 价格 |
|------|---------|-----------|----------|
| 标的价格 ↑ | 上涨 | **涨** | 跌 |
| 行权价 ↑ | 越高 | 跌 | **涨** |
| 到期时间 ↑ | 越长 | **涨** | **涨** |
| 波动率 ↑ | 越大 | **涨** | **涨** |
| 利率 ↑ | 越高 | 涨 | 跌 |

> 记忆要点：时间是买方的朋友（越久越可能赚钱），波动率越大期权越贵（不确定性本身值钱）。

### 实值、平值、虚值

```
以沪金期货 500 元/克为例：

Call 期权（看涨）：
  行权价 < 500 → 实值 ITM   （立即行权能赚钱）
  行权价 = 500 → 平值 ATM
  行权价 > 500 → 虚值 OTM   （行权不如直接买期货）

Put 期权（看跌）：
  行权价 > 500 → 实值 ITM   （立即行权能赚钱）
  行权价 = 500 → 平值 ATM
  行权价 < 500 → 虚值 OTM   （你的"吾剑策略"核心关注区）
```

### 期权价格的组成部分

```
期权价格 = 内在价值 + 时间价值

内在价值：
  Call：max(标的价格 - 行权价, 0)
  Put：max(行权价 - 标的价格, 0)
  虚值期权 → 内在价值 = 0

时间价值：
  期权价格 - 内在价值
  虚值期权的全部价格都是时间价值
  越临近到期 → 时间价值衰减越快
```

## 第 3 章：Greeks 入门

### 一句话定义

| Greek | 一句话 | 买方 | 卖方 |
|-------|--------|------|------|
| **Delta** | 标的涨 1 元，期权涨几元 | 正/负 | 反过来 |
| **Gamma** | 标的涨 1 元，Delta 变多少 | 正 | 负 |
| **Theta** | 过 1 天，期权掉几元 | **亏钱** | **赚钱** |
| **Vega** | 波动率涨 1%，期权涨几元 | 赚钱 | 亏钱 |
| **Rho** | 利率涨 1%，期权涨几元 | 微乎其微（商品期权忽略） | — |

### Delta：涨跌方向

```
Call Delta：0 到 1
Put Delta：-1 到 0

深度虚值 → Delta ≈ 0      （标的小动，期权几乎不动）
平值      → Delta ≈ ±0.5   （标的动 1 元，期权动 0.5 元）
深度实值 → Delta ≈ ±1      （标的动 1 元，期权动 1 元，像期货）
```

**对你卖方策略的意义**：卖的是深度虚值，Delta 很小。标的正常波动，浮亏很小。

### Gamma：Delta 的加速度

```
正 Gamma：标的有利 → 越赚越快（买方喜欢）
负 Gamma：标的不利 → 越亏越快（卖方)

平值 Gamma 最大。虚值 Gamma 较小。
临到期 Gamma 最大。远期 Gamma 较小。
```

**对你卖方策略的意义**：你卖的是深度虚值 + 有一定到期距离 → Gamma 够小。不是没风险，是风险不会突然爆炸。

### Theta：时间的朋友（卖方就是赚这个）

```
买方：每天亏 Theta → 时间是你的敌人
卖方：每天赚 Theta → 时间是你的朋友

平值 Theta 最大。虚值 Theta 较小。
临到期 Theta 最大。最后 30 天衰减加速。
```

**对你卖方策略的意义**：你卖虚值期权的核心利润来源。每天赚一点 Theta，到期归零赚全部。

### Vega：波动率敏感度

```
买方：Vega 正 → 波动率涨，你赚
卖方：Vega 负 → 波动率涨，你亏

平值 Vega 最大。虚值 Vega 较小。
远月 Vega 大于近月。
```

**对你卖方策略的意义**：你卖的时候希望 IV 高（贵卖）。卖完之后最怕 IV 暴涨（Vega 负，IV 涨你要亏）。

## 你卖虚值价差时的 Greeks 画像

```
头寸：卖豆粕 3400 Put，买 3350 Put

卖 3400 Put：Delta -0.15, Gamma -0.02, Theta +3.5, Vega -8
买 3350 Put：Delta +0.05, Gamma +0.01, Theta -1.5, Vega +3
────────────────────────────────────────────────
净头寸：    Delta -0.10, Gamma -0.01, Theta +2.0, Vega -5

翻译：
  • 标的跌 10 元，浮亏约 ¥1（Delta 小）
  • 每天赚约 ¥2 的 Theta（时间在你这边）
  • Vega -5：IV 暴涨你会浮亏，但信用价差封顶了最大亏损
```

## 必记的数字（直觉训练用）

| 情景 | Delta 大概多少 |
|------|---------------|
| 深度虚值（OTM 5%）| ±0.10-0.20 |
| 浅度虚值（OTM 2%）| ±0.25-0.35 |
| 平值 | ±0.50 |
| 实值 | ±0.65-1.00 |

| 情景 | Theta 特点 |
|------|-----------|
| 到期 > 60 天 | Theta 衰减慢 |
| 到期 30-60 天 | Theta 开始加速 |
| 到期 < 30 天 | Theta 衰减最快，但 Gamma 也大了 |
| 到期 < 7 天 | 极高 Theta + 极高 Gamma（风险最大） |

## 今日自检

能不看笔记说出这些就算通过：

1. 为什么虚值期权 Delta 小？→ 因为标的要移动很远才会进入实值，暂时对价格不敏感
2. 卖方靠什么赚钱？→ 靠 Theta（时间衰减）和 Vega 回落（IV 均值回归）
3. 卖方最大的两个风险是什么？→ 标的朝不利方向大幅移动 + IV 暴涨
4. 深度虚值期权的 Theta 比平值期权大还是小？→ 小，但胜率高。做的是概率，不是暴利
