## 今日学到

### 信用价差退出机制
- 持有到期（默认）：两条腿归零，赚全部权利金
- 提前平仓：净支出 < 净收入时，锁利润离场
- 防守平仓：期货跌近卖腿行权价 → 全平
- 展仓（Layer 2+）：到期前平当月，卖下月同款

### Call 信用价差
- 卖低行权价 Call + 买高行权价 Call = 不看涨
- 和 Put 信用价差完全对称，方向相反
- 两者可同时做 = 铁鹰策略（赌横盘不动）

### 期权策略四大方向
- 方向：买 Call/Put、牛市/熊市价差
- 波动率：跨式、宽跨式、信用价差
- 时间：信用价差、铁鹰、日历价差 ← 你现在在这里
- 结构：盒式价差、转换套利

### 卖方为什么叫 Writer
- 每一张期权都是从卖方"写出"来的合约，签上承诺"到期履约"
- 买方 = Holder（持有者），卖方 = Writer（写出者）

### 事件窗口不能做信用价差
- 信用价差卖方 = 卖波动率 = Vega 负
- 事件 = 波动率可能暴涨 → Vega 涨 = 浮亏
- 事件前适合买跨式（Vega 正），事件刚过 IV 回落适合卖方

### 止损
- 不看期权价，看标的期货价。期货流动性好，不会假突破
- 期权深度虚值区易触发假信号止损

### 机构零手续费
- 会员直连交易所 + 天量阶梯折扣 = 费用接近清算成本

### 50% 年化到 ¥10M
- 小资金可行（¥20K 月化 5% = ¥833/月）
- 瓶颈不是策略质量，是策略容量。¥5M 后信用价差挤不进去
- RenTec 70% 三要素：数据清洗极致 + 上万个弱信号叠加 + 封闭自有资金
