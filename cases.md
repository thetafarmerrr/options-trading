# 吾剑交易案例集 — 10个真实场景决策演练

> 每个案例按「数据 → 分析 → 决策 → 理由」四步展开
> 本金统一假设 10,000 元，品种沪金期权
> ⚠️ **看跌期权（Put）铁律**：行权价越高 → 价格越贵。**倒挂 = 行权价更高的合约反而更便宜**（如 P696 < P688）。P672 < P680 不是倒挂，是正常。

---

## 案例 1：完美信号

**时间**：上午 09:30，开盘半小时

**数据**：
```
P720: 买 1.44 / 卖 1.56   最新 1.50
P728: 买 0.68 / 卖 0.76   最新 0.72
──────────────────────────────────
倒挂！P728 行权价更高却更便宜 (0.72 < 1.50)
理论利润 = (1.50-0.72)/0.72 = 108%
买入 P728 @0.76（卖价）   卖出目标 @1.44（P720 买价）
扣除滑点净利 = (1.44-0.76)/0.76 = +89%
```

**分析**：
- 信号：P728 行权价 728 > P720 行权价 720，P728 应该更贵
- 但 P728(0.72) < P720(1.50) → P728 被低估，倒挂！
- 时间：处于最佳开仓窗口（09:00-10:15）
- 成本：0.76 × 1000 = 760 元/手
- 仓位：760 < 1000（10%上限），可买 1 手

**吾剑决策**：✅ 买入 P728 ×1 手 @0.76（被低估的那个）
- 目标价 1.50（回到 P720 的价格水平），止损 0.55
- 14:30 前不修复则离场

**理由**：信号强度高 + 时间充裕 + 仓位合规 + 成本低。

## 案例 2：滑点吃掉利润

**时间**：上午 10:00

**数据**：
```
P688: 买 4.60 / 卖 4.90   最新 4.80
P696: 买 3.68 / 卖 4.90   最新 4.44
──────────────────────────────────
倒挂！P696 行权价更高却更便宜 (4.44 < 4.80)
理论利润 = (4.80-4.44)/4.44 = 8.1%
买入 P696 @4.90（卖价）   卖出目标 @4.60（P688 买价）
扣除滑点净利 = (4.60-4.90)/4.90 = -6.1%
```

**分析**：
- P696 行权价更高 → 应该更贵，但 4.44 < 4.80 → 倒挂
- 理论利润 8.1%（按最新价算）很诱人
- 但买入 P696 要付卖价 4.90，卖出只能拿 P688 的买价 4.60
- 4.60 - 4.90 = -0.30 → **还没开始就亏 6.1%**
- 价差把理论利润吃得一干二净

**吾剑决策**：❌ 不做
- 再好的倒挂，价差吃掉利润就是陷阱
- 等做市商把价差收窄再考虑，或者等下一次机会

**理由**：买卖价差 > 理论利润 → 实际一买就亏。

## 案例 3：时间不够

**时间**：下午 14:20，距铁律 10 分钟

**数据**：
```
P704: 买 1.06 / 卖 1.14   最新 1.10
P720: 买 0.70 / 卖 0.78   最新 0.74
──────────────────────────────────
倒挂！P720 行权价更高却更便宜 (0.74 < 1.10)
理论利润 = (1.10-0.74)/0.74 = 48.6%
买入 P720 @0.78（卖价）   卖出目标 @1.06（P704 买价）
扣除滑点净利 = (1.06-0.78)/0.78 = +35.9%
```

**分析**：
- P720 行权价 720 > P704 行权价 704 → P720 应该更贵
- 但 P720(0.74) < P704(1.10) → 倒挂！P720 被低估
- 信号本身极强，净利 37%
- 成本 0.78 × 1000 = 780 元/手，仓位合规
- 但——只剩 10 分钟修复窗口

**吾剑决策**：❌ 不开仓
- 10 分钟修复概率太低
- 即使开仓，到 14:30 必须决定 → 大概率亏损离场
- 与其匆忙进场亏钱，不如错过

**理由**：14:00 后只做确定性最高的。这倒挂虽强，但 10 分钟不够。纪律 > 贪婪。

## 案例 4：仓位超限

**时间**：上午 09:45

**数据**：
```
P688: 买 4.60 / 卖 4.90   最新 4.80
P696: 买 3.90 / 卖 4.20   最新 4.44
──────────────────────────────────
倒挂！P696 行权价更高却更便宜 (4.44 < 4.80)
理论利润 = (4.80-4.44)/4.44 = 8.1%
买入 P696 @4.20（卖价）   卖出目标 @4.60（P688 买价）
扣除滑点净利 = (4.60-4.20)/4.20 = +9.5%
```

**分析**：
- P696 被低估（倒挂），且这次价差合理
- 理论净利 +9.5%，信号本身可做
- 但成本 4.20 × 1000 = 4,200 元/手
- 单笔上限 1,000 元 → 超限 4 倍

**吾剑决策**：❌ 不开仓
- 虽然信号可做，但 1 万本金撑不起 4200 元/手的成本
- 这不是「错过机会」，是「这个机会不属于现在的你」

**理由**：吾剑从 1035 元起步时，也只做几毛钱的权利金。等你本金到 5 万，这个信号就能做了。

## 案例 5：无信号日

**时间**：上午 09:00，盘前拉完数据

**数据**：
```
偏差检测结果：
  看跌倒挂: 0  看涨倒挂: 0
```

**分析**：
- 今天没有任何偏差
- 可能因为市场平静，做市商定价没有出错

**吾剑决策**：😴 等待，不交易
- 「大部分时间在等待」——这是原话
- 不交易 = 不亏钱 = 保住本金等明天的机会
- 用等待的时间做闪卡训练

**理由**：不做无效交易。没有偏差硬找机会，就是在赌博。

## 案例 6：连续三天倒挂缩小

> ⚠️ 注意：看跌期权，行权价越高越贵。倒挂 = 行权价更高的反而更便宜。

**时间**：上午 09:30，第三天的偏差数据

**数据**：
```
P688 > P696 倒挂趋势（P696 行权价更高，应该更贵，但实际更便宜）：
  Day1: P688(4.80) vs P696(4.20)  倒挂 14.3%  🆕
  Day2: P688(4.60) vs P696(4.30)  倒挂 7.0%   📉
  Day3: P688(4.40) vs P696(4.32)  倒挂 1.8%   📉

当前 P696: 买 4.28 / 卖 4.40  价差 2.8%
买入 P696 @4.40（卖价）→ 目标 P688 买价 4.20
净利 = (4.20-4.40)/4.40 = -4.5%
```

**分析**：
- P696 行权价 696 > P688 行权价 688 → P696 应该更贵
- 但 Day1 P696(4.20) < P688(4.80) → 倒挂 14.3%，P696 被低估
- 倒挂从 14.3% → 7.0% → 1.8%，连续缩小——做市商正在修复
- 但现在价差 2.8%，扣除后净利为负
- Day1 是最佳进场时机（买入被低估的 P696 @4.20，等回归）

**吾剑决策**：❌ 不做
- 倒挂快修复完了，利润空间不够
- 但——这个案例的价值是：验证了偏差确实会修复
- Day1 进场的人赚了 12%，现在该跑了

**理由**：不是每个偏差都要吃。错过了最佳进场点就等下一个。

## 案例 7：立刻修复

**时间**：上午 09:35 开仓，09:50 修复

**数据**：
```
09:30 发现: P720(1.50) > P728(0.72)  倒挂 108%  ← P728被低估
         P728: 买 0.68 / 卖 0.76
09:35 开仓: 买入 P728 @0.76 ×1手
09:50 修复: P728 买价涨到 1.10, 卖价 1.20
```

**分析**：
- 15 分钟就修复——做市商快速补了漏洞
- 卖出 P728 @1.10（买价）
- 利润 = (1.10 - 0.76) × 1000 = 340 元 (+45%)
- 目标已经超预期达成，不贪

**理由**：修复了就卖。不等「再多涨一点」。吾剑说「我要的是控制回撤，那就牺牲部分利润」。

## 案例 8：没修复，必须割

**时间**：上午 10:00 开仓，到 14:20 还没修复

**数据**：
```
10:00 发现: P696(0.74) > P704(0.62)  倒挂 ← P704 行权价更高却更便宜
         P696: 买 0.68 / 卖 0.78
         P704: 买 0.58 / 卖 0.62
10:00 开仓: 买入 P704 @0.62 ×1手
  目标 P696 买价 0.68
14:20 当前: P704 买价 0.54 / 卖价 0.58（不涨反跌）
```

**分析**：
- 偏差没修复，价格还跌了
- 浮亏 = (0.62 - 0.54) × 1000 = 80 元
- 还有 10 分钟到铁律

**吾剑决策**：🪓 主动降价卖出 P704 @0.50
- 实际亏损 = (0.62 - 0.50) × 1000 = 120 元
- 为什么挂 0.50 而不是 0.54？——确保有人接
- 挂 0.54 可能没人要，0.50 大概率成交

**理由**：「回撤超预期就主动降价，略微亏损也离场。要的是控制回撤，就牺牲部分利润。」40 元浮亏 vs 70 元实亏，多亏 30 但换来了确定性——不持仓过夜，不回撤扩大。

## 案例 9：14:30 后信号

**时间**：下午 14:35

**数据**：
```
刚拉完数据，发现：
P800: 买 2.50 / 卖 2.54   最新 2.54
P808: 买 2.28 / 卖 2.42   最新 2.32
──────────────────────────────────
倒挂！P808 行权价更高却更便宜 (2.32 < 2.54)
理论利润 = (2.54-2.32)/2.32 = 9.5%
买入 P808 @2.42（卖价）→ 目标 P800 买价 2.50
净利 = (2.50-2.42)/2.42 = +3.3%
```

**分析**：
- P808 被低估，信号质量不错：净利 +3.3%
- 但——14:35 了！铁律已过！

**吾剑决策**：🛑 不开仓，只记录
- 把信号记在交易日志里，标注「14:35 发现，未入场」
- 明天 08:30 优先关注这个偏差还在不在
- 如果明天还在且价差没扩大，才是真正的机会

**理由**：14:30 铁律不容妥协。

## 案例 10：周末发现信号

**时间**：周六上午，复盘时发现周五数据有信号

**数据**：
```
周五复盘发现：
P728: 买 1.44 / 卖 1.56   最新 1.50
P720: 买 0.68 / 卖 0.76   最新 0.72
──────────────────────────────────
倒挂！P720 行权价更高却更便宜
但周六才看到 → 无法交易

**分析**：
- 已经在周末，无法交易
- 但可以做一个验证：看下周一这个偏差是否修复了？

**吾剑决策**：📝 记录观察，周一看结果
- 如果周一修复了 → 验证信号有效 → 增强信心
- 如果周一还在 → 说明是持续性偏差 → 可能有更大利润空间
- 两种情况都是有价值的信息

**理由**：不交易的日子也是有价值的——用来验证策略、积累经验、训练判断力。
