## Day 1：看懂一张期权链

### 核心知识（45 min）

打开终端：
```bash
python3 fetch_options.py --variety au --contract au2608 --plain
```

你会看到一张表：
```
行权价  C最新  C买    C卖    P最新  P买    P卖    P持仓  信号
 664      -     -      -     0.40   0.40   0.42   831   深实
 672      -     -      -     0.48   0.34   0.88   105   深实
 ...
```

**理解每一列（不是抄，是对着看+回答）：**

- [ ] **行权价**：这是什么？谁定的？会变吗？
  - 交易所预设的、不会变的。好比赌场里固定的赔率档位。
- [ ] **C 最新价 / P 最新价**：这是哪里来的？
  - 市场撮合成交的最后价格。≠ 你现在买卖能拿到的价格。
- [ ] **C 买价 / P 买价**：你要**卖出**的话拿什么价格？
  - 买方愿意出的最高价。你卖给他 → 按买价成交。
- [ ] **C 卖价 / P 卖价**：你要**买入**的话付什么价格？
  - 卖方报的最低价。你从他手里买 → 按卖价成交。
  - **这是你真正的买入成本。不是最新价。**
- [ ] **持仓量**：这个合约有多少张还没平掉？
  - 越大 = 越多人参与 = 越容易进出。

### 速度训练（30 min）

```bash
python3 drill_system.py A    # 链面扫异常 — 第一次接触
```

今天不追求速度，先熟悉题型。

### 晚间记录（15 min）

写下今天的答案：
1. 沪金 au2608 最便宜的 Put 多少钱？在哪个行权价？
2. 最贵的 Put 多少钱？在哪个行权价？
3. 行权价从低到高，Put 价格从__到__？（递增还是递减？）

## Day 4：虚值和实值 — 为什么价格差那么大

### 核心知识（30 min）

读 Natenberg 第 1-2 章（期权基础 + 定价直觉）。

关键概念：

```
看跌期权（Put）：
  行权价 << 期货价 → 深度虚值 → 大概率不执行 → 极便宜
  行权价 ≈ 期货价 → 平值 → 一半一半 → 中等价格
  行权价 >> 期货价 → 深度实值 → 大概率执行 → 很贵
```

对照沪金链面：
- [ ] 深度虚值区在哪里？（行权价 < 期货价 - 80+）
- [ ] 平值在哪里？（行权价 ≈ 期货价）
- [ ] 为什么吾剑只买"几毛钱"的合约？在哪个区？

### 理解任务（15 min）

打开沪金链面，口头标记每行的"虚值/平值/实值"，不写下来。

### 速度训练（30 min）
